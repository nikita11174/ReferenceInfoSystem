# Система справочной информации

## Описание программы

Требуется разработать программу «Система справочной информации». Программа должна быть реализована с использованием технологий WPF (XAML).

### Основные требования

1. **Инициализация выпадающего списка**: 
   - При запуске программы должен быть проинициализирован выпадающий список с возможными типами устройств для выбора пользователем.
   - Данные для списка берутся по адресу:  
     `2392bb8b-2589-4515-a05d-bff3882c6c75.mock.pstmn.io/devices`

2. **Загрузка информации**: 
   - При выборе типа устройства из выпадающего списка необходимо нажать кнопку «Загрузить».
   - После нажатия кнопки программа должна отправить запрос по одному из следующих адресов и вывести информацию в табличной части программы:
     - `2392bb8b-2589-4515-a05d-bff3882c6c75.mock.pstmn.io/pumps`
     - `2392bb8b-2589-4515-a05d-bff3882c6c75.mock.pstmn.io/cylinders`
     - `2392bb8b-2589-4515-a05d-bff3882c6c75.mock.pstmn.io/valves`
   - Выводимые данные должны содержать:
     - Идентификатор
     - Обозначение
     - Наименование

3. **Отображение свойств оборудования**: 
   - Остальные свойства оборудования, специфичные для выбранного типа объекта, должны отображаться в отдельной панели свойств при выделении строки в табличной части программы.

4. **Интерфейс**: 
   - Программа должна поддерживать интерфейс на английском и русском языках с «горячим» переключением языка.
   - Кнопка для переключения языка должна отображать текущий выбранный язык.
